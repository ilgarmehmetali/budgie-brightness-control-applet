# Budgie Brightness Control Applet
This applet allows you to controll screen brightness.

---

## Dependencies
```
budgie-1.0 >= 2
gnome-desktop-3.0
gnome-settings-daemon
gtk+-3.0 >= 3.18
glib-2.0
libpeas-1.0 >= 1.8.0
vala
```

### Installing

**From source**  
```bash
mkdir build && cd build
meson --prefix /usr --buildtype=plain ..
ninja
sudo ninja install
```
